> How come it needs 256megs of RAM?

It actually needs less. There are three boot options:

-	Full 32 bit RAMD.  The RAM Disk size is ~ 190 MB
	Compressed: 64 MB

-	Root only 32 bit RAMD. The RAM Disk size is ~ 64 MB
	Compressed: 19 MB

-	Root only 32/64 bit RAMD. The RAM Disk size is ~ 110 MB
	Compressed: 35 MB

Solaris will boot of Grub + gzip compressed RAMD + uncompressed RAMD
fit into core. The smalles Boot Option will work with 128 MB of RAM.

If I have some time, I may move some of the drivers from /kernel
to /usr/kernel and reduce the small root from 64 MB to ~ 30 MB.
Then it may be possible to boot with 64 MB of RAM.
